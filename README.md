# Baggo's WhatsApp Bot

Bot de WhatsApp con IA para Baggo's Baguettería en José Cardel, Veracruz.

## Características

- 🤖 IA conversacional con Claude
- 📱 WhatsApp Business integration
- 🍔 Menú completo de Baggo's
- 📦 Gestión automática de pedidos
- 🔔 Notificaciones a WhatsApp personal
- 📄 Generación de comandas

## Deploy en Render

1. Fork este repositorio
2. Crear Web Service en Render
3. Conectar con este repositorio
4. Escanear QR que aparece en logs
5. ¡Listo!

## Variables de Entorno (Opcional)

- `CLAUDE_API_KEY`: Tu API key de Claude (ya incluida temporalmente)
- `PORT`: Puerto (lo asigna Render automáticamente)
